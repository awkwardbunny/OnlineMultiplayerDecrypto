# Decrypto

Online version of the [Decrypto](https://boardgamegeek.com/boardgame/225694/decrypto) board game

## Running the game

Run the backend on one terminal

```bash
cd backend
python3 setup.py install
python3 -m src
cd ..
```

Then launch the frontend on another

```bash
cd frontend
yarn install
yarn start
cd ..
```
