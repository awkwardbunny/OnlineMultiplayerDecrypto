# Decrypto

Online version of the [Decrypto](https://boardgamegeek.com/boardgame/225694/decrypto) board game

## Running the game

First run the backend

```bash
cd backend
python3 setup.py install
python3 src # ??? idk
cd ..
```

Then launch the frontend

```bash
cd frontend
yarn install
yarn start
cd ..
```
